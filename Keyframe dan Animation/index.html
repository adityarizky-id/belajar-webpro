Keyframe

CSS dapat mengubah halaman web Anda menjadi sebuah karya seni, tetapi Anda tidak dapat menyebut elemennya interaktif kecuali mereka dapat mendeteksi perubahan di browser.

JavaScript menampilkan keindahan dari *keyframe* yang sesungguhnya. Anda dapat menggunakannya untuk mengaktifkan animasi pada titik tertentu atau memulainya dalam keadaan tertentu.
